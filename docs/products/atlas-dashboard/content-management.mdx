# Gestión de contenido

La pantalla **Contenido** (`/admin/content`) es el inventario de tu catálogo. El encabezado muestra el total de
ítems y cuántos se agregaron esta semana, y una tabla lista cada ítem con búsqueda y filtros.

## Encontrar ítems

- **Búsqueda**: cuadro de texto libre para encontrar ítems por título.
- **Filtro por tipo**: Libro, Video, Episodio, En vivo, Podcast, Artículo.
- **Filtro por estado**: Borrador, Programado, Publicado, Archivado.

La tabla muestra la miniatura y el título de cada ítem, su tipo, estado (con código de color), conteo de lecturas de 30 días y
hora de última actualización, con un enlace **Editar** por fila.

## Acciones en lote

Selecciona filas con las casillas para **Publicar**, **Archivar** o **Eliminar** varios ítems a la vez
(Eliminar requiere el permiso correspondiente). También puedes **Exportar CSV** para una copia sin conexión, y
gestionar tus **Tipos de contenido** (clave, etiqueta, habilitado) en un cuadro de diálogo.

## Crear un ítem

Haz clic en **Nuevo** para abrir el formulario de creación:

| Campo | Notas |
|---|---|
| Título | Obligatorio |
| Slug | Opcional; se deriva automáticamente del título (minúsculas, guiones) |
| Tipo | book, video, episode, live, podcast, article |
| Estado inicial | draft, scheduled, published |
| Metadatos | Datos estructurados opcionales (p. ej. idioma) |

Después de guardar, puedes subir una imagen de portada desde la página de edición.

## Editar un ítem

El editor usa pestañas para que cada sección se guarde de forma independiente: **Básicos** (título, slug, tipo, estado),
**Media** (subidas de portada/póster), **Reparto y equipo**, **SEO**, **Disponibilidad** y
**Publicación** (programación). Una barra lateral muestra la portada, el estado actual y las métricas de interacción. Si
otro operador edita el mismo ítem al mismo tiempo, verás una advertencia de edición concurrente para que no se pierda ningún trabajo.
