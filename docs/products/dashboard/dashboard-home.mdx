# Dashboard home

The dashboard home is the first screen after sign-in. It summarizes everything happening across your workspace.

## Stat cards

Four cards across the top give you the headline numbers:

| Card | What it shows |
|---|---|
| **Articles published** | Content published this month, with the change vs. last month. |
| **Credits used (MTD)** | Credits consumed month-to-date, with the trend arrow. |
| **Active domains** | How many domains are currently connected. |
| **Open invoices** | Total value of invoices still outstanding. |

An up or down arrow next to a number tells you whether it grew or shrank compared with the previous month.

## Credits usage trend

A chart shows usage over the last 12 weeks. Switch between the **Credits**, **Articles**, and **Invoices** tabs to change the metric. Below the chart you'll see the weekly average and the peak week.

## Credits card

On the right, the credits card shows your current balance and plan label. From here you can **Top up credits** or jump to your full usage history. See [Billing & credits](/docs/products/dashboard/billing-credits).

## Recent activity

A live feed lists recent events in your workspace. Choose **View all** to open the full activity log (admin workspaces). See [Activity & request logs](/docs/products/dashboard/admin-logs).

## Quick actions

Shortcut buttons let you jump straight to common tasks: start new content, issue an invoice, add a domain, or top up credits. If activity or stats are still loading you'll see placeholders; if there's nothing yet, a friendly empty state explains what will appear here once you start using the workspace.
