# Catálogo y taxonomía

La pantalla **Taxonomía** (`/admin/taxonomy`) controla cómo se agrupa y se navega tu catálogo. Estas
configuraciones alimentan las páginas de género, autor y editorial en tu storefront público.

## Géneros

Una tabla repetible define tus géneros. Cada fila tiene:

| Columna | Propósito |
|---|---|
| Slug | El segmento de URL para el género (p. ej. `fantasy`) |
| Etiqueta | El nombre visible mostrado a los visitantes |
| Intro HTML | Texto introductorio opcional para la página del género |

La última fila siempre está en blanco para que puedas agregar un nuevo género; las filas que se dejen vacías se ignoran al guardar.

## Introducciones

Dos áreas de texto enriquecido agregan textos introductorios a tus páginas de listado:

- **Intro de autores**: aparece en la página índice de autores.
- **Intro de editoriales**: aparece en la página índice de editoriales.

Estas ayudan a los visitantes (y a los motores de búsqueda) a entender cada sección de un vistazo.

## Enlaces cruzados

La sección **Enlaces cruzados** te permite conectar secciones relacionadas de tu catálogo; por ejemplo, vincular un
género con una agrupación de autores correspondiente. Se edita como una lista estructurada de asignaciones origen/destino.

## Guardar

Haz clic en **Guardar taxonomía** para aplicar. Los cambios se resuelven por host, por lo que tu storefront
los refleja sin necesidad de un redespliegue.

El título del catálogo y las plantillas usadas para los títulos de las páginas de taxonomía se encuentran en
[Configuración SEO](/docs/products/atlas-dashboard/seo-configuration). Para ver dónde aparecen estas páginas
públicamente, lee [Páginas del storefront](/docs/products/atlas-dashboard/storefront-pages).
