# Settings — Workspace

The **Workspace** tab lives under Settings, alongside Profile, Security, Billing, Notifications, API, and Team.

## Coming soon

This pane is a **planned feature**. When it ships, it will bring workspace-level identity settings into one place, including:

- **Workspace name** — the display name for your workspace.
- **Logo** — your workspace logo.
- **Default business** — the issuer used by default for invoicing.
- **Workspace identifier** — the tenant identifier for your workspace.

Until then, the tab shows a friendly "coming soon" message so you know exactly what to expect.

## In the meantime

Some of these settings are already available to workspace administrators elsewhere:

- The workspace **display name** and security keys are managed on the admin App settings screen — see [Admin & impersonation](/docs/products/dashboard/admin-impersonation).
- Branding such as logo, colors, and name is applied per tenant at runtime — see [Branding & appearance](/docs/products/dashboard/branding-appearance).
- The **default business** for invoicing is configured under [Invoicing businesses](/docs/products/dashboard/invoicing-businesses).

If you need to change any of these before the Workspace tab ships, contact your workspace administrator or support.
