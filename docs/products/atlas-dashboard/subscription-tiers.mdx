# Niveles de suscripción

Dos pantallas cubren las membresías: **Planes** (`/admin/plans`) define lo que vendes, y
**Suscripciones** (`/admin/subscriptions`) muestra quién está suscrito.

## Planes

La pantalla Planes resume tu cantidad de planes, el total de suscriptores y los ingresos recurrentes. Cada plan es una
tarjeta que puedes habilitar, deshabilitar o eliminar. Para crear uno, completa el formulario de agregar plan:

| Campo | Notas |
|---|---|
| Nombre | El nombre visible del plan |
| Precio | El monto cobrado |
| Moneda | La moneda del precio |
| Período | Único, Mensual o Anual |
| Funciones | Un beneficio por línea |
| URL de pago | Dónde los compradores completan el pago |

Cada tarjeta también muestra el pago neto después de las comisiones de la plataforma y un conteo de suscriptores en vivo, para que puedas ver
cómo se desempeña cada nivel de un vistazo.

## Suscripciones

La pantalla Suscripciones es tu panel de ingresos recurrentes. Las tarjetas de resumen muestran los ingresos recurrentes
mensuales, los suscriptores activos, los miembros en prueba y el abandono de 30 días. La tabla de abajo lista cada
suscriptor con su plan, estado (**Activo**, **Prueba**, **Vencido**, **Cancelado**), fecha de alta
y contribución a los ingresos.
