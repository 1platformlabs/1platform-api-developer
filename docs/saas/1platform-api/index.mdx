# 1Platform API

La **1Platform API** es la API REST central que impulsa cada producto de 1Platform. Construye tu propio
ecosistema sobre ella: generación de contenido con IA, palabras clave y SEO, imágenes, pagos y suscripciones,
facturación electrónica, gestión de dominios, agentes de IA y más — a través de una única API unificada y versionada.

> Esta sección es para **desarrolladores**. Para los productos finales configurables, consulta
> [Productos (por tenant)](/docs/products/dashboard/overview).

## Autenticación — dos tokens

Cada solicitud lleva dos tokens:

| Token | Header | Identifica |
|---|---|---|
| **Token de aplicación** | `Authorization: Bearer ak-...` | Tu aplicación |
| **Token de usuario** | `x-user-token: sk-...` | El usuario final que actúa |

```bash
curl https://api.1platform.pro/api/v1/... \
  -H "Authorization: Bearer ak-your-app-token" \
  -H "x-user-token: sk-the-user-token"
```

Consulta [Primeros pasos](/docs/saas/1platform-api/getting-started) para el recorrido completo de la primera llamada,
y [Entornos](/docs/saas/1platform-api/reference/environments) para las URLs base.

## Qué puedes construir

Pagos con tarjeta y suscripciones, facturación electrónica, contenido e imágenes generados con IA, investigación
de palabras clave y SEO, gestión de sitios web y páginas legales, dominios personalizados, agentes de IA, registros de
actividad y programas de referidos — todo bajo un único par de claves de API, con sobres de respuesta y webhooks consistentes.
