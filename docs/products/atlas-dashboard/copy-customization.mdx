# Personalización de textos

La pantalla **Textos** (`/admin/content/copy`) te permite sobrescribir el texto visible en tu storefront:
etiquetas de botones, encabezados, marcadores de posición y prosa editorial, para que toda la tienda hable con la voz de tu
marca. Cualquier campo que dejes en blanco vuelve a un valor localizado por defecto sensato, de modo que solo editas lo que
quieres cambiar.

## Cómo funciona

Cada campo muestra una etiqueta humanizada, el texto por defecto como marcador de posición y tu sobrescritura actual. Escribe
tu propia redacción para reemplazar el valor por defecto; borra el campo para volver al valor por defecto. Cuando termines,
haz clic en **Guardar textos**.

## Secciones que puedes editar

Los campos están agrupados por área de la tienda para que sean fáciles de encontrar:

| Grupo | Ejemplos de lo que controlas |
|---|---|
| Navegación | Etiquetas de menú y navegación |
| Inicio y carruseles | Encabezados de inicio y etiquetas de carrusel |
| Catálogo y búsqueda | Etiquetas del catálogo, marcador del buscador |
| Próximamente | Mensajería previa al lanzamiento / no disponible |
| Detalle del ítem | Etiquetas en la página del ítem |
| VIP | Redacción premium / de membresía |
| Géneros | Etiquetas de la página de género |
| Acceso | Avisos de inicio de sesión y cuenta |
| Legal y pie de página | Enlaces legales y texto del pie de página |
| FAQ y prosa editorial | Bloques de texto de formato largo |
| Estados vacíos | Mensajes cuando una lista no tiene ítems |
