# Configuración — Workspace

La pestaña **Workspace** se encuentra en Configuración, junto a Perfil, Seguridad, Cobros, Notificaciones, API y Equipo.

## Próximamente

Este panel es una **función planificada**. Cuando se lance, reunirá la configuración de identidad a nivel del workspace en un solo lugar, incluyendo:

- **Nombre del workspace** — el nombre visible de tu workspace.
- **Logo** — el logo de tu workspace.
- **Empresa predeterminada** — el emisor usado por defecto para la facturación.
- **Identificador del workspace** — el identificador de tenant de tu workspace.

Hasta entonces, la pestaña muestra un mensaje amigable de "próximamente" para que sepas exactamente qué esperar.

## Mientras tanto

Algunos de estos ajustes ya están disponibles para los administradores del workspace en otros lugares:

- El **nombre visible** del workspace y las claves de seguridad se gestionan en la pantalla de administración Configuración de la app — consulta [Administración e impersonación](/docs/products/dashboard/admin-impersonation).
- La marca, como el logo, los colores y el nombre, se aplica por tenant en tiempo de ejecución — consulta [Marca y apariencia](/docs/products/dashboard/branding-appearance).
- La **empresa predeterminada** para la facturación se configura en [Empresas de facturación](/docs/products/dashboard/invoicing-businesses).

Si necesitas cambiar alguno de estos antes de que se lance la pestaña Workspace, contacta a tu administrador del workspace o a soporte.
