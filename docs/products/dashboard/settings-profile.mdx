# Settings — Profile

The **Profile** tab (under Settings) controls how you appear across the workspace. It's the default tab when you open Settings.

## What you can edit

| Field | Notes |
|---|---|
| **Profile photo** | Upload a JPG or PNG up to 2 MB. Square images work best. |
| **Full name** | The name others see across the platform. |
| **Email** | Your contact email address. |
| **Job title** | Your role or title. |
| **Language** | Choose your interface language (Spanish or English). |

## Updating your photo

1. Select **Upload photo**.
2. Pick a square JPG or PNG up to 2 MB.
3. The dashboard confirms the photo is ready.

## Saving changes

After editing any field, select **Save changes** — a confirmation message appears. To undo edits you haven't saved, select **Cancel** and the form returns to its last saved values.
