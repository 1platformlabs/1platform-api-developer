# Promotional Rails

The **Home Rails** screen (`/admin/rails`) controls the carousels ("rails") that make up your
storefront home page. The order here matches the order visitors see, so this is where you curate the
first impression of your store.

## Adding a rail

Use the add form to create a new rail:

| Field | Notes |
|---|---|
| Title | The heading shown above the carousel |
| Type | Top 10, Continue watching, By genre, New releases, or Custom |
| Add | Appends the rail to the list |

## Arranging rails

Each rail in the ordered list shows its type badge, an inline-editable title, move up/down buttons, an
enable/disable toggle, a delete button, and its position (e.g. "1st"). Reorder rails to control the
flow of the page, and disable a rail to hide it temporarily without deleting it.

## Rail types

- **Top 10** — a ranked highlight row.
- **Continue watching** — resumes content for returning visitors.
- **By genre** — pulls from a genre in your [taxonomy](/docs/products/atlas-dashboard/catalog-and-taxonomy).
- **New releases** — your most recent items.
- **Custom** — a curated row you control.

If no rails are configured, an empty state prompts you to add your first one.

The home page itself is described in
[Storefront Pages](/docs/products/atlas-dashboard/storefront-pages), and the wording around rails can
be adjusted in [Copy Customization](/docs/products/atlas-dashboard/copy-customization).
