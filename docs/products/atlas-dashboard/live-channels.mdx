# Live Channels

If your store offers live streaming, the **Live** screen (`/admin/live`) is your control room for
monitoring and toggling live events. It appears only when the live capability is enabled for your
store, and starting or stopping events requires the right permission.

## At a glance

Four summary cards give you the current picture:

| Card | Meaning |
|---|---|
| Live now | Events currently broadcasting |
| Viewers | Total viewers across all live events |
| Scheduled | Upcoming events |
| Channels | Number of available channels |

## Event cards

Below the summary, each event appears as a card showing its title, status (**Live**, **Scheduled**,
or **Ended**), the channel it broadcasts on, current viewer count, and start time. A button lets you
**Start** a scheduled event or **Stop** a live one; ended events are read-only.

## Programme guide (EPG)

The **EPG** screen (`/admin/epg`) shows a schedule grid — channels down the side, hourly time slots
across the top. The current hour is highlighted with a **Now** marker, and cells that are airing are
tinted with a **Live** badge.

If you maintain your schedule in a standard programme-guide file, use the **Import** action to upload
it. You'll see a confirmation when the guide updates, or an error prompt to retry. This screen also
respects the live capability and the import permission.

The public-facing live pages are described in
[Storefront Pages](/docs/products/atlas-dashboard/storefront-pages).
