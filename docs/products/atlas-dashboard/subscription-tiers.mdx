# Subscription Tiers

Two screens cover memberships: **Plans** (`/admin/plans`) defines what you sell, and
**Subscriptions** (`/admin/subscriptions`) shows who is subscribed.

## Plans

The Plans screen summarizes your plan count, total subscribers, and recurring revenue. Each plan is a
card you can enable, disable, or delete. To create one, fill in the add-plan form:

| Field | Notes |
|---|---|
| Name | The plan's display name |
| Price | The amount charged |
| Currency | The currency for the price |
| Period | One-time, Monthly, or Yearly |
| Features | One benefit per line |
| Checkout URL | Where buyers complete payment |

Each card also shows the net payout after platform fees and a live subscriber count, so you can see
how each tier performs at a glance.

## Subscriptions

The Subscriptions screen is your recurring-revenue dashboard. Summary cards show monthly recurring
revenue, active subscribers, members in trial, and 30-day churn. The table below lists each
subscriber with their plan, status (**Active**, **Trial**, **Past due**, **Canceled**), join date,
and contribution to revenue.
