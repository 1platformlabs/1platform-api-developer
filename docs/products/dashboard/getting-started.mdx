# Primeros pasos

El dashboard es el panel de control de tu workspace. Esta página cubre el inicio de sesión y cómo orientarte.

## Inicia sesión con un enlace mágico

No hay contraseñas. Para iniciar sesión:

1. Abre el dashboard de tu workspace en el navegador.
2. Ingresa la dirección de correo invitada al workspace.
3. Revisa tu bandeja de entrada para encontrar un enlace de inicio de sesión de un solo uso.
4. Haz clic en el enlace — entrarás directamente al dashboard.

El enlace es de un solo uso y tiene tiempo limitado. Si expira, simplemente solicita uno nuevo. Como no hay contraseña, no hay nada que almacenar, restablecer ni filtrar.

## Qué verás primero

Después de iniciar sesión llegas al inicio del **Dashboard**, que te saluda por tu nombre y muestra tu workspace de un vistazo: saldo, uso de este mes, dominios activos, facturas abiertas y actividad reciente.

La barra lateral izquierda lista los módulos habilitados para tu workspace. Las entradas comunes incluyen Dashboard, Cobros, Facturación y Dominios, además de Configuración y (para administradores) Registros y Usuarios. Qué entradas aparecen, y en qué orden, se configura por workspace — consulta [Módulos](/docs/products/dashboard/modules).

## Finalizar la configuración

Los workspaces nuevos muestran una lista de verificación de onboarding que te guía para conectar un dominio, generar tu primer contenido, configurar la facturación e invitar a tu equipo. Puedes seguirla paso a paso u omitirla y volver más tarde. Consulta [Onboarding](/docs/products/dashboard/onboarding).

## La barra superior

Según la configuración de tu workspace, la barra superior puede incluir un selector de idioma, un interruptor de tema claro/oscuro y una campana de notificaciones. Tu menú de cuenta te permite llegar a tu perfil y cerrar sesión.
