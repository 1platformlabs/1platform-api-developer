# Configuración — Perfil

La pestaña **Perfil** (en Configuración) controla cómo apareces en todo el workspace. Es la pestaña predeterminada cuando abres Configuración.

## Qué puedes editar

| Campo | Notas |
|---|---|
| **Foto de perfil** | Sube un JPG o PNG de hasta 2 MB. Las imágenes cuadradas funcionan mejor. |
| **Nombre completo** | El nombre que otros ven en toda la plataforma. |
| **Correo** | Tu dirección de correo de contacto. |
| **Puesto** | Tu rol o cargo. |
| **Idioma** | Elige el idioma de tu interfaz (español o inglés). |

## Actualizar tu foto

1. Selecciona **Subir foto**.
2. Elige un JPG o PNG cuadrado de hasta 2 MB.
3. El dashboard confirma que la foto está lista.

## Guardar cambios

Después de editar cualquier campo, selecciona **Guardar cambios** — aparece un mensaje de confirmación. Para deshacer ediciones que no has guardado, selecciona **Cancelar** y el formulario vuelve a sus últimos valores guardados.
