# Facturación

La página **Facturación** te permite emitir y dar seguimiento a facturas electrónicas para tu workspace.

## Las tarjetas de resumen

Cuatro tarjetas te dan las cifras principales del periodo que estás viendo:

| Tarjeta | Significado |
|---|---|
| **Emitidas (mes en curso)** | Facturas emitidas en lo que va del mes, con el cambio respecto al mes anterior. |
| **Total facturado** | El valor combinado de las facturas en la ventana seleccionada. |
| **Pendiente** | Valor de las facturas enviadas pero aún no liquidadas. |
| **Vencidas** | Cantidad de facturas que pasaron su fecha de vencimiento. |

## Encontrar facturas

La tabla de facturas se puede acotar de tres maneras:

1. **Pestañas de estado** — Todas, Borrador, Enviada, Pagada, Vencida, Cancelada.
2. **Buscar** — por número de factura, cliente o monto.
3. **Rango de tiempo** — últimos 30 días, últimos 90 días, este mes o este año.

Un pie de página muestra cuántas facturas están visibles y la ventana de fechas activa.

## Emitir una nueva factura

Selecciona **Nueva factura** para comenzar. Primero eliges la empresa emisora, así que asegúrate de tener al menos una configurada — consulta [Empresas de facturación](/docs/products/dashboard/invoicing-businesses). Si aún no tienes ninguna, el estado vacío te enlaza directamente a la pantalla de Empresas para configurar tu primer emisor.

## Exportar

Usa **Exportar CSV** para descargar las facturas que se muestran actualmente (que coincidan con tus filtros y búsqueda) como un archivo de hoja de cálculo. La exportación respeta las pestañas, el texto de búsqueda y el rango de tiempo que estén activos, de modo que obtienes exactamente la porción que estás viendo.

## Estados que verás

Las facturas pasan por *Borrador*, *Enviada*, y pueden terminar como *Cancelada*. La pestaña de estado que elijas filtra la tabla para mostrar solo esas facturas.
