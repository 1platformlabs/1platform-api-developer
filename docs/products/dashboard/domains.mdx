# Dominios

La página **Dominios** te permite conectar, gestionar y monitorear los dominios usados en tu workspace — las direcciones donde se publica tu contenido.

## La tabla de dominios

Cada dominio conectado aparece como una fila que muestra:

- **Dominio** — el nombre de dominio.
- **Nota / registrador** — dónde se gestiona el dominio.
- **Estado** — su estado actual (ver abajo).
- **Renueva** — la próxima fecha de renovación, cuando se conoce.

## Estados de dominio

| Estado | Qué significa |
|---|---|
| **Activo** | El dominio está conectado y funcionando. |
| **DNS pendiente** | El dominio se está configurando o transfiriendo y aún no está en línea. |
| **Expirado** | El dominio ha caducado y necesita atención. |

## Agregar un dominio

Selecciona **Agregar dominio** para conectar o registrar uno nuevo. Cuando aún no tienes dominios, un estado vacío te invita a conectar o registrar tu primer dominio para que puedas empezar a publicar.

## Si la lista no carga

Si los dominios no se pueden obtener, la página muestra un error con un botón **Reintentar** — selecciónalo para intentar cargar de nuevo. Mientras se obtienen los datos aparecen brevemente marcadores de posición de carga.

## Dónde encaja esto

Los dominios forman parte de la configuración de tu workspace. La lista de verificación de onboarding te anima a conectar tu primer dominio cuanto antes — consulta [Onboarding](/docs/products/dashboard/onboarding) — y el inicio del dashboard cuenta tus dominios activos como una estadística principal.
