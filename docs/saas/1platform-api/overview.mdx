# 1Platform API

The **1Platform API** is the core REST API that powers every 1Platform product. Build your own
ecosystem on top of it: AI content generation, keywords and SEO, images, payments and subscriptions,
electronic invoicing, domain management, AI agents, and more — through one unified, versioned API.

> This section is for **developers**. For the configurable end-products, see
> [Products (per tenant)](/docs/products/dashboard/overview).

## Authentication — two tokens

Every request carries two tokens:

| Token | Header | Identifies |
|---|---|---|
| **App token** | `Authorization: Bearer ak-...` | Your application |
| **User token** | `x-user-token: sk-...` | The end user acting |

```bash
curl https://api.1platform.pro/api/v1/... \
  -H "Authorization: Bearer ak-your-app-token" \
  -H "x-user-token: sk-the-user-token"
```

See [Getting started](/docs/saas/1platform-api/getting-started) for the full first-call walkthrough,
and [Environments](/docs/saas/1platform-api/reference/environments) for base URLs.

## What you can build

Card payments and subscriptions, electronic invoicing, AI-generated content and images, keyword
research and SEO, website and legal-page management, custom domains, AI agents, activity logs, and
referral programs — all under one API key pair, with consistent response envelopes and webhooks.
