# Compras y alquileres

La pantalla **Alquileres / PPV** (`/admin/rentals`) sigue las compras individuales: eventos de pago por evento y
alquileres por tiempo limitado, para tiendas que venden ítems individuales en lugar de (o además de)
membresías.

Esta pantalla está disponible solo cuando las compras individuales están habilitadas para tu tienda. Si está desactivada,
verás una explicación en lugar del panel.

## De un vistazo

Cuatro tarjetas de resumen te dan las cifras destacadas:

| Tarjeta | Significado |
|---|---|
| Alquileres activos | Alquileres actualmente dentro de su ventana de visualización |
| Ingresos PPV | Ingresos por compras individuales |
| Precio prom. | Precio promedio en todas las ventas |
| Conversiones | Cuántos visitantes completaron una compra |

## Tabla de ventas

La tabla lista cada ítem vendible con:

- **Título** y su insignia de tipo de contenido.
- **Modelo**: **PPV** (pago por evento) o **Alquiler**.
- **Precio**: el monto cobrado.
- **Ventana de visualización**: cuánto dura el acceso (por ejemplo, "Evento" o "48 horas").
- **Ventas**: cuántas veces se ha vendido.
