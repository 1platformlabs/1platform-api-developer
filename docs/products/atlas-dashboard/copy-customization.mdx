# Copy Customization

The **Copy** screen (`/admin/content/copy`) lets you override the visible text across your storefront —
button labels, headings, placeholders, and editorial prose — so the whole store speaks in your brand
voice. Anything you leave blank falls back to a sensible localized default, so you only edit what you
want to change.

## How it works

Each field shows a humanized label, the default text as a placeholder, and your current override. Type
your own wording to replace the default; clear the field to return to the default. When you're done,
click **Guardar textos** (Save texts).

## Sections you can edit

Fields are grouped by area of the store so they're easy to find:

| Group | Examples of what you control |
|---|---|
| Navigation | Menu and nav labels |
| Home & Rails | Home headings and carousel labels |
| Catalog & Search | Catalog labels, search placeholder |
| Coming soon | Pre-launch / unavailable messaging |
| Item detail | Labels on the item page |
| VIP | Premium / membership wording |
| Genres | Genre page labels |
| Access | Sign-in and account prompts |
| Legal & Footer | Legal links and footer text |
| FAQ & Editorial prose | Long-form copy blocks |
| Empty states | Messages when a list has no items |
