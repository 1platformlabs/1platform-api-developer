# 1Platform Dashboard

El **1Platform Dashboard** es el panel de control white-label que tu workspace usa para operar
todo lo que 1Platform impulsa — cobros y créditos, facturación, dominios personalizados, equipo y roles,
claves API, y un conjunto modular de funciones que activas o desactivas por tenant.

Cada tenant ejecuta el mismo producto pero obtiene su **propia marca**: logo, colores, tipografía, nombre,
módulos habilitados y feature flags se resuelven en tiempo de ejecución, de modo que el dashboard se ve y se comporta como
*tu* producto, no como una herramienta genérica.

> Esta sección es para **operadores** que configuran el producto para un workspace — sin código necesario.
> Si quieres integrar de forma programática, consulta la [1Platform API](/docs/saas/1platform-api/overview).

## Qué puedes hacer

| Área | Qué hace un operador |
|---|---|
| **Dashboard** | Ver las estadísticas del workspace de un vistazo — saldo, uso de este mes, actividad reciente. |
| **Cobros y créditos** | Consultar el saldo, recargar créditos, revisar el historial completo de transacciones. |
| **Facturación** | Emitir y dar seguimiento a facturas por estado, y gestionar las empresas emisoras. |
| **Dominios** | Agregar y gestionar los dominios personalizados de tu workspace. |
| **Equipo y roles** | Invitar miembros, asignar roles, desactivar cuentas. |
| **Claves API** | Generar y rotar las claves que usan las integraciones. |
| **Configuración** | Perfil, workspace, seguridad y preferencias de notificaciones. |
| **Módulos** | Habilitar módulos opcionales — sitios web, herramientas, plugins, tareas, tickets, preguntas frecuentes, referidos. |

## White-label por diseño

Lo que puedes personalizar por tenant:

- **Marca** — nombre, logo (claro/oscuro), favicon, imagen social, correo de soporte, enlaces legales.
- **Apariencia** — paleta de colores, tipografía, superficies claras, estilo de componentes.
- **Módulos** — qué funciones aparecen en la barra lateral, su orden y quién puede acceder a ellas.
- **Feature flags** — despliegue progresivo de capacidades por workspace.

El inicio de sesión es **solo por enlace mágico** — los miembros reciben un enlace de un solo uso por correo; no hay contraseña que
gestionar.
