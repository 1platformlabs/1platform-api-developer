# ¿Qué API necesito?

1Platform Labs expone dos APIs SaaS independientes. Elige la que coincida con lo que estás construyendo y luego
abre su referencia interactiva para explorar y probar endpoints en vivo.

| API | Construye… | URL base | Referencia |
|---|---|---|---|
| **1Platform API** | Contenido con IA, pagos, facturación, dominios, agentes — la plataforma de producto completa. | `https://api.1platform.pro/api/v1` | [Abrir →](/api-reference/1platform-api) |
| **Atlas API** | Un ecosistema multitenant de entrega de contenido — catálogo, derechos, entrega, webhooks. | `https://atlas-api.1platform.pro/api/v1` | [Abrir →](/api-reference/atlas-api) |

## 1Platform API

La API central de la plataforma. Autenticación de **dos tokens** (`Authorization: Bearer ak-...` +
`x-user-token: sk-...`). Empieza con la [vista general](/docs/saas/1platform-api/overview) y los
[primeros pasos](/docs/saas/1platform-api/getting-started).

- [Referencia interactiva →](/api-reference/1platform-api)

## Atlas API

Una plataforma multitenant e independiente de entrega de contenido. Autenticación con **JWT bearer** más
resolución por tenant; independiente de la 1Platform API. Empieza con la
[vista general](/docs/saas/atlas-api/overview).

- [Referencia interactiva →](/api-reference/atlas-api)

:::note
Las referencias interactivas renderizan la especificación OpenAPI de cada API. Su contenido permanece en el
idioma de origen de la API; el portal que las rodea está disponible en inglés y español.
:::
